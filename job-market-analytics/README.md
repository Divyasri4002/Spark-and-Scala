# Job Market Data Analysis (Streamlit + PySpark)

## Objective
Analyze job listings to identify skill demand, hiring trends, and salary insights.

## Tech Stack
- Python
- PySpark
- Streamlit
- Plotly

## How to Run

1. Install dependencies
   pip install -r requirements.txt

2. Start the app
   streamlit run app.py

3. Open Spark UI
   http://localhost:4040

## Features
- Interactive filters
- Real-time Spark analytics
- Stunning UI
- Spark Web UI integration
